Scratch Projects

By Mikael

Scratch is a beginners coding language that uses blocks to create lines of code. There are many things that the program is capable of, even on an outdated school computer. It employs the use of coordinates to detect the location of entites called sprites, which are movable computer graphics, like a playable character in a video game. You can use hundreds of blocks to control these sprites. The most important these are the sensor blocks. They each detect one thing on the computer. For example, a line of code for this would be: If [space key pressed] move sprite to x-14,y-243. The sensor block would detect when the space key is pressed, and it would execute the command, moving the blocks to the chosen coordinates. You should definitely check out the website. You can:

  1. Use hundreds of blocks to code sprites, or create your owm
  2. Share or publish your code with millions of people worldwide, like a social media website
  3. freely acess millons of templates, helping you to create your own
  4. create your own sprites, sound effects, and even games
